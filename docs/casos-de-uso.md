# Casos de Uso — StelLoot

---

## Caso de Uso: Login

### Ator
Usuário

### Fluxo Principal
1. Usuário acessa a tela de login
2. Digita email e senha
3. Sistema valida os dados
4. Usuário acessa a plataforma

---

## Caso de Uso: Buscar Jogos

### Ator
Usuário

### Fluxo Principal
1. Usuário digita o nome do jogo
2. Sistema consulta a API
3. Jogos encontrados são exibidos

---

## Caso de Uso: Favoritar Jogo

### Ator
Usuário

### Fluxo Principal
1. Usuário abre detalhes do jogo
2. Clica em favoritar
3. Sistema salva nos favoritos

---

## Caso de Uso: Criar Alerta

### Ator
Usuário

### Fluxo Principal
1. Usuário define valor desejado
2. Sistema monitora preço
3. Sistema envia alerta

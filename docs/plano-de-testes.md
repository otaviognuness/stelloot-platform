# Plano de Testes — StelLoot

## Objetivo

Garantir o funcionamento correto das funcionalidades principais da plataforma StelLoot.

---

# Funcionalidades Testadas

## 1. Login

### Cenário
Usuário realiza login com email e senha válidos.

### Resultado Esperado
Sistema deve permitir acesso à plataforma.

---

## 2. Cadastro

### Cenário
Usuário cria uma nova conta.

### Resultado Esperado
Conta deve ser registrada no banco de dados.

---

## 3. Busca de Jogos

### Cenário
Usuário pesquisa um jogo pelo nome.

### Resultado Esperado
Sistema deve retornar jogos encontrados pela API.

---

## 4. Favoritos

### Cenário
Usuário adiciona um jogo aos favoritos.

### Resultado Esperado
Jogo deve aparecer na lista de favoritos.

---

## 5. Alertas de Preço

### Cenário
Usuário cria alerta de preço para um jogo.

### Resultado Esperado
Sistema deve armazenar o alerta corretamente.

---

# Tipos de Teste

- Testes unitários
- Testes de integração
- Testes funcionais

---

# Ferramentas Utilizadas

- JUnit

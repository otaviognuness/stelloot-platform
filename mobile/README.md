# StelLoot Mobile

Aplicativo mobile da StelLoot.

Esta pasta deve concentrar o app React Native/Expo quando o desenvolvimento mobile for integrado ao repositorio.

## Responsabilidades

- Consumir a mesma API do backend usada pelo front web.
- Exibir ofertas de jogos.
- Permitir wishlist e alertas de preco.
- Manter experiencia adaptada para celular.

## Stack prevista

- React Native
- Expo
- Consumo da API Spring Boot em `backend/`
